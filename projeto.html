1- O que é o HTML

(voltar)

O HTML (Hypertext Markup Language) é a linguagem de marcação fundamental usada para criar páginas web na internet. Ele fornece a estrutura básica de qualquer página, definindo os elementos como texto, imagens, links, tabelas e outros conteúdos.

2- Como surgiu ?

(voltar)

Foi inventado por Tim Bernes Lee em 1991, junto com a criação do www (word wide web) , com a ideia principal de criar uma maneira de simples de estruturar documentos e compartilha-los por meio da rede. Berners-Lee propôs usar hiperlinks para conectar diferentes documentos de texto. Isso levou à criação de uma linguagem de marcação (markup language) que permitia o uso de links entre diferentes páginas na web o HTML.

Desde então, o HTML evoluiu bastante, passando por várias versões até chegar ao HTML5, que suporta vídeos, animações e muito mais. Mas a base da internet ainda é a mesma: páginas escritas em HTML!

3- O primeiro navegador gráfico - Mosaic

(voltar)

Mosaic, lançado em 1993 por uma equipe da Universidade de Illinois, antes dele a maioria dos navegadores eram baseados em texto, como o Lynx. O Mosaic foi o primeiro navegador a:

  • Exibir imagens e textos juntos na mesma página.
  • Ter uma interface gráfica amigável, facilitando o uso.
  • Introduzir botões de navegação (Voltar, Avançar, Atualizar).

Isso popularizou a World Wide Web (WWW), tornando a navegação acessível ao público e inspirou a criação do Netscape Navigator (1994), que dominou o mercado antes da chegada do Internet Explorer, sem o Mosaic, a web poderia ter demorado muito mais para se tornar visual e interativa como conhecemos hoje.

4- HTMl e seu uso em navegadores web

(voltar)

O HTML é a linguagem padrão usada para estruturar o conteúdo de páginas web, que por sua vez, tem a função de ajudar os navegadores,(Chrome, Firefox, Edge, Safari, etc.) que são os programas responsáveis por interpretar e exibir esse código para o usuário de forma visual.

Como os navegadores processam o HTML?

  1. O navegador recebe o código HTML de um servidor web.
  2. Ele analisa (parseia) o HTML e constrói a estrutura da página.
  3. Baixa e aplica arquivos CSS para estilizar o conteúdo.
  4. Executa scripts JavaScript para tornar a página interativa.
  5. Renderiza a página na tela do usuário.

Padrões Web e Compatibilidade

Os navegadores seguem padrões definidos pelo W3C (World Wide Web Consortium) e pelo WHATWG. Cada navegador pode interpretar certos recursos de maneira ligeiramente diferente, por isso testar compatibilidade é essencial para o código de sites bonitos e funcionais!.

5- Principais elementos mais utilizados

(voltar)
  • HTML – Elemento raiz Define o início e o fim de um documento HTML.
  • HEAD – Cabeçalho da página Contém metadados sobre o documento, como título, links para arquivos CSS e scripts.
  • TITLE – Título da página Define o título exibido na aba do navegador.
  • BODY - Corpo da página Contém o conteúdo visível da página, como textos, imagens, links e outros elementos.
  • H1,H2,H3...H6– Cabeçalhos Usados para títulos e subtítulos, de H1 (mais importante) até H6 (menos importante).
  • P -Paragrafo Usado para definir blocos de texto
  • a – Link (âncora) Usado para criar hyperlinks que levam a outras páginas ou recursos.
  • Ul-lista ordenada(bullets)
  • Ol-lista não ordenada (numerada)
  • li- item de lista

    • Eses são só alguns elementos usados no HTML que possibilitam criar páginas web, juntos eles compoẽm a estrutura básica que permite a criação de um conteudo interativo e visualmente estruturado.
