1- O que é o HTML

(voltar)

O HTML (Hypertext Markup Language) é a linguagem de marcação fundamental usada para criar páginas web na internet. Ele fornece a estrutura básica de qualquer página, definindo os elementos como texto, imagens, links, tabelas e outros conteúdos.

2- Como surgiu ?

(voltar)

Foi inventado por Tim Bernes Lee em 1991, junto com a criação do www (word wide web) , com a ideia principal de criar uma maneira de simples de estruturar documentos e compartilha-los por meio da rede. Berners-Lee propôs usar hiperlinks para conectar diferentes documentos de texto. Isso levou à criação de uma linguagem de marcação (markup language) que permitia o uso de links entre diferentes páginas na web o HTML.

3- Principais elementos mais utilizados

(voltar)
  • HTML – Elemento raiz Define o início e o fim de um documento HTML.
  • HEAD – Cabeçalho da página Contém metadados sobre o documento, como título, links para arquivos CSS e scripts.
  • TITLE – Título da página Define o título exibido na aba do navegador.
  • BODY - Corpo da página Contém o conteúdo visível da página, como textos, imagens, links e outros elementos.
  • H1,H2,H3...H6– Cabeçalhos Usados para títulos e subtítulos, de H1 (mais importante) até H6 (menos importante).
  • P -Paragrafo Usado para definir blocos de texto
  • a – Link (âncora) Usado para criar hyperlinks que levam a outras páginas ou recursos.
  • Ul-lista ordenada(bullets)
  • Ol-lista não ordenada (numerada)
  • li- item de lista

    • Eses são só alguns elementos usados no HTML que possibilitam criar páginas web, juntos eles compoẽm a estrutura básica que permite a criação de um conteudo interativo e visualmente estruturado.
